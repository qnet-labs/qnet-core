### Added
- Initial release: quantum network entanglement distribution simulator
- BBPSSW fidelity purification protocol
- Event-driven simulation engine with binary heap scheduler
- Three routing strategies: lowest latency, highest fidelity, highest success rate
- Monte Carlo ensemble simulation support
- Pre-built topology generators (telecom backbone, repeater chain, hybrid satellite-fiber)
- Topology comparison and diffing tools
- .qnet file format for saving/loading network topologies
- PyO3 Python bindings with comprehensive API
